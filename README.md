# Bob's Git Utilities

I think this speaks for itself.

## Dependencies

You'll need [git-extras](https://github.com/tj/git-extras) and [moz-git-tools](https://github.com/mozilla/moz-git-tools).

## Installation

On Linuxbrew or Homebrew:

```bash
% brew tap rwhogg/bob
% brew install bobs-git-utils
```
